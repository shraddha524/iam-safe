# iam-safe appication
I Am Safe – Women’s Safety Mobile App 
Tech Stack: Android Studio, Java, XML, Google Maps API, GPS

"I Am Safe" is an Android mobile application designed to enhance women’s safety by providing real-time emergency assistance. Key features include:

Instant SOS alerts to pre-selected contacts or authorities

Real-time GPS tracking using Google Maps API

User-friendly interface for quick access during distress

40% reduction in emergency response time through fast notifications

The app empowers users to feel safer and connected, especially in critical situations.

 Features

Instant SOS Alerts
  Send emergency messages to pre-selected contacts or local authorities with a single tap.

Real-Time GPS Tracking 
  Share your live location using Google Maps API for quick and accurate response.

User-Friendly Interface 
  Simple and intuitive UI to ensure fast access during emergencies.

Quick Notification System
  Reduced emergency response time by 40% with fast and automated alerts.


  Requirements
Android Studio (latest version recommended)

Java (JDK 8 or higher)

Google Maps API Key

Android Device with GPS enabled
